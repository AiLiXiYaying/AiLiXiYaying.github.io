这一节讲IDE（集成开发环境，Integrated Development Environment），例如市场上目前主流的IDE：

  • Visual Studio 20xx
  • Visual Studio Code
  • Hbuilder

还有很多IDE,就不一一列举了。

那么现在一定有人疑惑，IDE有什么用呢?

IDE的作用为：

  1. 提供调试工具
  2. 提供代码编辑
  3. 集成编译器和构建工具
  4. 插件支持
  5. 集成测试工具

认识完IDE后，那我现在就要开始推荐了，对于目前时代而言，微软旗下的VS系列，必然而然是最优选择，在排除掉特别因素的情况下，VCODE（Visual Studio Code简称）基本可以干任何事情，但不适合处理大型工程项目。接下来由我细细的介绍微软旗下的IDE。

先从VCODE开始讲吧，VCODE适用于轻量级项目，一般是供个人开发或小团体开发使用，因其插件繁多而闻名，可编写超多种语言，至少目前市场上你能看到的编程语言，基本都可以使用vcode配合插件进行编写代码。其次的就是顺畅感，人见人爱。

剩下的就是VS(Visual Studio 20xx)了,这个IDE主要是用于大型工程项目，适用于大团队进行开发使用，并且程序开发环境只需要点几下手指就可以配置好。优点说完了，那就说一下缺点，太过于臃肿，对于某些语言，编译项目复杂。所以综上所述，仅适合团体使用。如果是个人开发者或小项目开发者，老老实实用VCODE吧，用习惯之后，真的会爱上她。
